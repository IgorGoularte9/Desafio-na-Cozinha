 Desafio na CozinhaDisciplina: Algoritmos e Estruturas de Dados II Trabalho: 1    Sobre o ProjetoEste é um sistema inteligente de gerenciamento de receitas e menus. O projeto foi desenvolvido para organizar o acervo culinário de um restaurante, permitindo buscas rápidas, geração de recomendações e a verificação de integridade para garantir que as receitas não sejam adulteradas.
